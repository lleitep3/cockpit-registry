# code-gif

Equips the AI with the ability to generate Mac-styled terminal GIFs directly from text inputs using Python.

## Description

This package deploys the `code-gif` skill to your AI agents, allowing them to render beautiful terminal-like animations featuring a dark background, monospace text, and a Mac OS title bar layout (with red, yellow, and green buttons).

## Usage

Once installed, simply ask your AI agent:
"Gere um gif animado do terminal executando o comando X e Y."

The agent will use the deployed Python script to generate the GIF and save it locally.
